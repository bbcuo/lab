Java 基本数据类型

Java基础 · 2026-07
▼
Java 中有 8 种基本数据类型（primitive types），分为四类：整型、浮点型、字符型和布尔型。
📄 数据类型定义示例
public class DataTypeDemo {
    public static void main(String[] args) {
        // 整型
        byte b = 127;           // 1字节，-128 ~ 127
        short s = 32767;        // 2字节
        int i = 2147483647;   // 4字节（最常用）
        long l = 9223372036854775807L; // 8字节，注意加 L

        // 浮点型
        float f = 3.14f;      // 4字节，注意加 f
        double d = 3.1415926535; // 8字节（默认）

        // 字符型
        char c = 'A';          // 2字节，Unicode

        // 布尔型
        boolean flag = true;  // 只有 true/false

        System.out.println("int 最大值: " + i);
    }
}

Java 流程控制语句

Java基础 · 2026-07
▼
Java 流程控制分为：顺序结构、分支结构（if-else, switch）、循环结构（for, while, do-while）。
📄 if-else 和 switch 示例
int score = 85;

// if-else 分支
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 80) {
    System.out.println("良好");
} else {
    System.out.println("继续努力");
}

// switch 分支（JDK 14+ 支持箭头语法）
String day = "MON";
switch (day) {
    case "MON" -> System.out.println("星期一");
    case "FRI" -> System.out.println("星期五");
    default  -> System.out.println("其他");
}
📄 循环语句示例
// for 循环：打印九九乘法表
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= i; j++) {
        System.out.print(j + "×" + i + "=" + (i * j) + "\t");
    }
    System.out.println();
}

// while 循环
int count = 0;
while (count < 5) {
    System.out.println("count = " + count);
    count++;
}

Java 数组的定义与使用

Java基础 · 2026-07
▼
数组是存储同类型多个元素的容器。Java 中数组一旦创建，长度不可变。常用操作：遍历、排序、查找。
📄 数组基本操作
import java.util.Arrays;

public class ArrayDemo {
    public static void main(String[] args) {
        // 数组的三种定义方式
        int[] arr1 = new int[5];               // 动态初始化
        int[] arr2 = {1, 2, 3, 4, 5};       // 静态初始化
        int[] arr3 = new int[]{1, 2, 3};      // 完整写法

        // 遍历数组
        for (int i = 0; i < arr2.length; i++) {
            System.out.print(arr2[i] + " ");
        }
        System.out.println();

        // 增强 for 循环（for-each）
        for (int num : arr2) {
            System.out.print(num + " ");
        }

        // 使用 Arrays 工具类
        System.out.println(Arrays.toString(arr2));  // [1, 2, 3, 4, 5]
        Arrays.sort(arr2);                         // 排序
    }
}

}

}

Git 常用命令速查

工具 · 2026-07
▼
Git 是分布式版本控制系统，掌握这些命令可以高效管理代码版本。
💻 Git 基本工作流
# 初始化仓库
git init

# 配置用户信息
git config user.name "bbcuo"
git config user.email "your-email@example.com"

# 查看状态
git status

# 添加文件到暂存区
git add .                    # 添加所有文件
git add index.html           # 添加指定文件

# 提交
git commit -m "初始化项目"

# 查看提交历史
git log --oneline

# 分支操作
git branch                   # 查看分支
git branch feature-xxx       # 创建分支
git checkout feature-xxx     # 切换分支
git merge feature-xxx        # 合并分支

# 远程仓库
git remote add origin https://github.com/bbcuo/repo.git
git push -u origin main      # 推送到远程
git pull origin main         # 拉取远程更新

MySQL 基础 SQL 语句

数据库 · 2026-07
▼
SQL（结构化查询语言）是操作关系型数据库的标准语言。CRUD 是最基本的四种操作。
💻 CRUD 基本操作
-- 创建数据库
CREATE DATABASE mydb;
USE mydb;

-- 创建表
CREATE TABLE student (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(50) NOT NULL,
    age INT,
    score DECIMAL(5, 2)
);

-- 插入数据 (Create)
INSERT INTO student (name, age, score)
VALUES ('张三', 20, 88.5),
       ('李四', 21, 92.0);

-- 查询数据 (Read)
SELECT * FROM student;
SELECT name, score FROM student WHERE score > 90;
SELECT * FROM student ORDER BY score DESC;

-- 更新数据 (Update)
UPDATE student SET score = 95 WHERE name = '张三';

-- 删除数据 (Delete)
DELETE FROM student WHERE id = 1;
